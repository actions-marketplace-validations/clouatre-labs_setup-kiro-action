# Security Policy

## Reporting Vulnerabilities

**Email:** hugues@linux.com (or use GitHub Security Advisory)

**Scope:** This action only (not Kiro CLI)

**Response:** 48 hours acknowledgment, 7-14 days fix

## Supported Versions

| Version | Supported |
|---------|-----------|
| v1.x    | Yes       |
| < v1.0  | No        |

## Out of Scope

- Kiro CLI bugs: https://github.com/aws/amazon-q-developer-cli/security
- Workflow security: See [examples/](examples/)
